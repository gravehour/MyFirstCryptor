# Мой первый криптор на C++

Это мой первый проект на C++ для шифрования текста. Программа использует алгоритм XOR для шифрования.

## Как использовать

1. **Подготовьте файл для шифрования**: Добавьте любой текст в файл `text.txt` (он уже есть в репозитории для примера).
2. **Соберите проект**: Запустите `make` в корне проекта. Это создаст исполняемый файл в папке `bin/`.
3. **Запустите шифрование**: Выполните `./bin/cryptor text.txt`. Программа автоматически:
   - Зашифрует содержимое `text.txt`.
   - Сохранит зашифрованные данные в `data.bin`.
   - Сохранит ключ шифрования в `key.bin`.
   - Расшифрует данные обратно и сохранит в `decrypted.txt` (для проверки корректности).
4. **Результат**: Файлы `data.bin` и `key.bin` — это зашифрованный файл и ключ. `decrypted.txt` должен совпадать с оригиналом.

**Важно**: Ключ (`key.bin`) генерируется случайно при каждом запуске. Храните его в безопасном месте для возможности расшифровки. Для реального использования модифицируйте код, чтобы загружать существующий ключ вместо генерации нового.
