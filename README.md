# Мой первый криптор на C++

Это мой первый проект на C++ для шифрования текста. Программа использует алгоритм XOR для шифрования.

## Как использовать

1. Добавьте любой текст в файл `text.txt`.
2. Соберите проект с помощью `make`.
3. Запустите `./bin/cryptor` для шифрования.

Зашифрованный текст будет сохранён в `bin/data.bin`, а ключ — в `bin/key.bin`. Для расшифровки используйте тот же ключ.
